flask-dashboard
Minimal Flask web dashboard with sample product metrics.

Overview  
This project demonstrates a lightweight dashboard built using Flask.  
It displays key business metrics such as user activity, revenue, and retention  
through a simple HTML table rendered directly from a Python dictionary.

Features  
- Flask web framework  
- Inline HTML templating with dynamic data rendering  
- Example business KPIs displayed in tabular format  
- Simple, portable architecture  

Tools  
Python • Flask • HTML • CSS  

Usage  
Run the app locally:

Visit `http://127.0.0.1:5000` in your browser to view the dashboard.
